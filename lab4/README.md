# Векторный графический редактор (Qt)

Практическое задание №4 по Qt: простейший векторный редактор с обработкой
событий мыши, объектной моделью графических примитивов и сериализацией сцены.

## Сборка и запуск

**Qt Creator:** откройте `VectorEditor.pro` (Qt 5.15+/6.x) и нажмите Run.

**CMake:**
```bash
mkdir build && cd build
cmake ..
cmake --build .
./VectorEditor
```

## Структура проекта

| Файл | Назначение |
|---|---|
| `shape.h/.cpp` | Базовый класс `Shape`: цвет контура/заливки, толщина линии, выделение, точки, сериализация, фабрика |
| `shapes.h/.cpp` | 6 производных классов: `LineShape`, `RectangleShape`, `EllipseShape`, `CircleShape`, `PolygonShape`, `FreehandShape` |
| `scene.h/.cpp` | Коллекция объектов: хранение, hit-test, порядок слоёв, сериализация в файл (`.vge`) |
| `canvas.h/.cpp` | Рабочая область: `paintEvent`, обработка мыши (создание/выделение/перемещение/resize), undo/redo, copy/paste, zoom |
| `mainwindow.h/.cpp` | Меню, панель инструментов (выбор фигуры, цвета, толщина линии) |
| `tests/` | Модульные тесты (QtTest) для `Shape`/`Scene`, отдельный исполняемый файл |
| `sample_data/` | Пример сохранённой сцены |
| `screenshots/` | Скриншоты работы приложения |

## Реализованный функционал

- Рабочая область на `QPainter` + `paintEvent()`.
- Мышь: нажатие / перемещение / отпускание — создание фигур перетаскиванием;
  двойной клик и правая кнопка — завершение многоугольника и контекстное меню объекта.
- 6 типов примитивов: линия, прямоугольник, эллипс, окружность, многоугольник, кривая.
- Объектная модель: базовый класс `Shape` + производные классы, коллекция в `Scene`.
- Выделение кликом (hit-test с учётом заливки/контура), перемещение перетаскиванием,
  изменение размера через ручки на границе (для линии — за оба конца).
- Изменение цвета контура/заливки и толщины линии — как для новых, так и для
  уже выделенных объектов.
- Удаление объектов, очистка холста.
- Сериализация всей сцены в бинарный файл `.vge` (`QDataStream`, с заголовком-магией
  и версией формата) — сохранение и загрузка с обработкой ошибок повреждённого файла.

### Дополнительные возможности

- **Undo/redo** — стек снимков сцены (переиспользует бинарную сериализацию).
- **Копирование/вставка** — через тот же механизм сериализации объекта.
- **Изменение порядка слоёв** — на передний/задний план, поднять/опустить.
- **Масштабирование рабочей области** — Ctrl+колесо мыши или меню «Вид».

## Модульные тесты

`tests/tst_scene.cpp` — 12 тестов для `Shape`/`Scene` (создание объектов, hit-test
для разных фигур, удаление, порядок слоёв, перемещение, сериализация/файлы,
обработка повреждённых и отсутствующих файлов). Собираются в отдельный
исполняемый файл `VectorEditorTests`, независимый от GUI.

\`\`\`bash
cd build
ctest --output-on-failure
\`\`\`

Результат: 14 passed, 0 failed.

## Проверка через реальное взаимодействие мышью

Помимо unit-тестов, приложение проверено через симулированные события мыши
(создание фигур перетаскиванием, выделение, перемещение, удаление, undo,
копирование/вставка, масштабирование) — скриншоты каждого шага в screenshots/.
